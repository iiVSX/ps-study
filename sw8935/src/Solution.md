Algorithm : DP
Problem : 정렬된 배열 A에서 순서 상관 없이 B를 배치할 때, 나열된 물체를 연속으로 선택할 수 없을 때 고를 수 있는 물체들의 합의 최댓 값을 구하시오
Idea : 
if n-1번째 물건을 선택한 경우
  if n번째 물건이 A배열인 경우
    A배열의 현재 물건을 버린다
  if n번째 물건이 B배열인 경우
    B배열의 현재 물건을 버린다
if n-1번째 물건을 선택하지 않은 경우
  if n번째 물건이 A배열인 경우
    A배열의 현재 물건을 선택한다
    A배열의 현재 물건을 버린다
  if n번째 물건이 B배열인 경우
    B배열의 현재 물건을 선택한다
    B배열의 현재 물건을 버린다
